JPO - Projet évenementiel de groupe

Cahier des Charges

  • Durée: 16 heures
  • Créer une activité(un jeu) pour des lycéens n’y connaissant quasiment rien au code
  • Livrables
    • 1 démo de jeu coder en Java
    • 1 sujet PDF imprimmable

Outils Utilisés

  • Netbeans pour réaliser le code Java du jeu
  • Gitlab pour manager le projet
  • Microsoft Word pour écrire le sujet du jeu

Compétences Acquises

  • Travail organisé en équipe
  • Organisation d'un dépôt GitLab à 3
  • Travail de "vulgarisation" de l'apprentissage du langage Java

Résultat

Interface graphique de la démo de jeu
